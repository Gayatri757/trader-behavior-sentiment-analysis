Trader Performance vs Market Sentiment Analysis
Objective

To analyze how market sentiment (Fear vs Greed) impacts trader profitability and behavior on Hyperliquid.

Methodology

Cleaned and prepared sentiment and trader datasets

Converted timestamps and aligned data at daily level

Merged datasets on date

Created metrics:

Daily PnL

Win rate

Trade frequency

Long/Short bias

Segmented traders by frequency and consistency

Key Insights

Trader profitability shows variation across Fear and Greed market conditions.

Profit volatility increases during Greed periods.

Frequent traders maintain more stable performance across sentiment cycles.

Strategy Recommendations

Reduce position size during Fear periods to manage downside risk.

Adjust trading frequency dynamically based on sentiment strength.

How to Run
pip install -r requirements.txt

Open analysis.ipynb and run cells sequentially.

Keep it clean and short.
